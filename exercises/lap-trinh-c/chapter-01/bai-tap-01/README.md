# Bài tập 1: Hello World

## Đề bài

Viết chương trình in ra màn hình dòng chữ "Hello, World!".

## Yêu cầu

1. Sử dụng hàm `printf` từ thư viện `stdio.h`
2. Chương trình phải kết thúc bằng lệnh `return 0;`
3. Compile và chạy thành công

## Input

Không có input

## Output

```
Hello, World!
```

## Ví dụ

```
Output:
Hello, World!
```

## Lưu ý

- Đây là bài tập đầu tiên để làm quen với C
- Chú ý dấu chấm phẩy `;` ở cuối câu lệnh
- Dấu nháy kép `"` bao quanh chuỗi
